Use Numerical_Sim to write code for the state-space models.
The parameters of the aircraft can be accessed by typing <AIRCRAFT_NAME>.<TYPE_OF_PROPERTIES>.<DESIRED_PARAMETER>
e.g. I want CLa of the Cessna aircraft: cessna.AeroProp.CLa
If you do not know the exact name of the type of properties or desired parameter you can use TAB to search through a list (Spyder).
